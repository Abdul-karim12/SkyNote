# SkyNote

Cloud-based notes API with authentication and user-specific data.

## Stack
Flask • SQLite • Docker • AWS EC2 • Terraform

## Features
- User registration and login
- Token-based authentication
- Private notes per user
- Full CRUD API
- Persistent storage (Docker volume)
- Deployed on AWS EC2

## Focus
Backend • Auth • Database • Docker • Cloud
